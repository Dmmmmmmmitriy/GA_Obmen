# Генетический алгоритм оптимизации

## Задание
Найти максимум функции **f(x) = 3x³ - 2x + 5** на интервале **[1, 10]**

## Вариант 14
- **I(A)**: Бинарное кодирование хромосом
- **II(A,C)**: Стратегии "одеяла" и фокусировки
- **III(A,B)**: Случайная селекция и селекция по шкале
- **IV(A,B,I)**: Одноточечный, двухточечный и циклический кроссинговер
- **V(A,F)**: Простая мутация и инверсия
- **VI(B)**: Элитный отбор
- **VII(A)**: Микроэволюция
